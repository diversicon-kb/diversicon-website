About

Simple Java library to provide a multilingual SQL knowledge base for lexicons and concepts, based upon UBY.

Features:

  • thin wrapper over UBY, adds:
    - transitive closure computation for SynsetRelations
    - simplified API for compressed XML I/O (in LMF-UBY format) and SQL backup
    - a testing framework to quickly create graphs via api
    - more metadata about DB and imports
    - improved logging
    - Ships with Wordnet 3.1 dump
  • access to DB via Hibernate
  • for now fully supports H2 DB, a pure Java database (both in-memory and on disk). No separate installation is required.
  • dependency handling with Maven
  • unit tested with proper integration tests
  • supports Java 7+

Dependencies

  • DKPRO UBY
  • Hibernate
  • H2 database
  • Apache Commons IO and Compress, XZ Archiver
  • Diversicon Model Diversicon Model

Projects using Diversicon

  • S-Match UBY: a connector for S-match semantic framework
