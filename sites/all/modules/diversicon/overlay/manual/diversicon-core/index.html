About Core

Java library that provides a multilingual SQL knowledge base for lexicons and concepts, based upon DKPRO UBY. Part of Diversicon tools.

UsageLicenseRoadmapContributing

Features

  • Improves DKPRO UBY with stronger validation, namespaces handling and I/O management
  • access to DB via Hibernate
  • based on H2 DB, a pure Java database (both in-memory and on disk). No separate installation is required.
  • dependency handling with Maven
  • unit tested with proper integration tests
  • supports Java 7+

Dependencies

  • DKPRO UBY
  • Hibernate
  • H2 database
  • Apache Commons IO and Compress, XZ Archiver
  • Diversicon Model

Projects using Diversicon Core

  • DiverCLI: a command line tool to manage Diversicons
  • DivMaker: XML LMF converter
  • S-Match Diversicon: a connector for S-match semantic framework
