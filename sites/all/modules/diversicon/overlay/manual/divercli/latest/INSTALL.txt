

						DIVERCLI INSTALLATION
	
	
  RUNNING
	
    After you unzipped the archive, you can run the software straight away without 
    installing anything:
	
      on Windows
	  
         1) click Start
         2) in the Search or Run line type   cmd   and press enter.  	
         3) cd  to the directory where divercli is unzipped and type:
	   
             bin\divercli.bat	
	
      on Mac/Linux    
	  	
         open a terminal and type:
	
             bin/divercli
	  	    
  RUNNING FROM ANY DIRECTORY
	
    To be able to run divercli easily from any directory, after you unzipped somewhere the archive, 
    you can add the divercli  'bin/'  directory to your path like so:
